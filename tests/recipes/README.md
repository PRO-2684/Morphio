# Example Recipes

- [`coverage.toml`](tests/recipes/coverage.toml): Primarily for testing, covering all branches of morphing process.
- [`latex_unicode.toml`](tests/recipes/latex_unicode.toml): Recipe for rendering LaTeX math expressions with Unicode characters. You may have to enable "Skip missing glyphs" if your font doesn't support all the characters.
- [`simple.toml`](tests/recipes/simple.toml): Simple example recipe to render `Microsoft` as `Microslop`.
